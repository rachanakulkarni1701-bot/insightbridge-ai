# InsightBridge

**InsightBridge** is an AI-powered Profit and Growth Decision Intelligence Agent designed to support executive decision-making.

## Overview
InsightBridge analyzes revenue, order volume, gross margin, and key cost drivers to determine whether growth is profitable and sustainable.
